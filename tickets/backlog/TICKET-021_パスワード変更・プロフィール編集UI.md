## 概要

CLAUDE.md のロール定義に「`user` は自身の認証情報を変更できる」と定義されているが、対応するフロントエンド UI が存在しない。ログイン中のユーザーが自分のパスワードと表示名を変更できる画面を実装する。

---

## 背景・目的

管理者（admin）がユーザーを作成した後、一般ユーザーが自分でパスワードを変更できる手段がない。初期パスワードから安全なパスワードへの変更を可能にすることはセキュリティ上も必要。

---

## 受け入れ条件

- [ ] ナビゲーションバーからプロフィール編集画面に遷移できる
- [ ] `display_name` を変更できる
- [ ] 現在のパスワード・新しいパスワード・確認用パスワードを入力してパスワードを変更できる
- [ ] 現在のパスワードが誤っている場合はエラーメッセージを表示する
- [ ] 新しいパスワードと確認用が一致しない場合はバリデーションエラーを表示する
- [ ] 対応するバックエンド API が未実装の場合は先に実装する

---

## サブチケット（コミット単位）

- [ ] `feat(api): パスワード変更・表示名変更エンドポイントを openapi.yaml に追加`（未実装の場合）
