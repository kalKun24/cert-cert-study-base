## 概要

コメントに親子関係（1 階層のみ）と @メンション機能を追加する。
Slack スタイルのスレッドを想定し、返信を親コメントの下に展開表示する。

---

## 背景・目的

既存のコメントは単一フラットリスト。スレッドとメンションにより議論の文脈が整理しやすくなる。

---

## 受け入れ条件

- [ ] コメントエンティティに `parent_id: string | null` フィールドが追加されている（1 階層のみ。親コメントへの返信のみ可）
- [ ] `mentions: []string`（メンション対象ユーザーID一覧）フィールドが追加されている
- [ ] `POST /api/v1/questions/{id}/comments` が `parent_id` を受け付ける
- [ ] コメント取得 API がスレッド構造（親 + 子リスト）でレスポンスを返す
- [ ] フロントエンド: 親コメントに「返信」ボタンがあり、クリックで返信フォームが展開される
- [ ] フロントエンド: 親コメントの下に返信を折りたたみ/展開表示できる
- [ ] フロントエンド: コメント本文に `@username` を入力するとメンバー候補がオートコンプリートされる
- [ ] `api/openapi.yaml` が更新されている
- [ ] ユースケース層のユニットテストが作成されている
