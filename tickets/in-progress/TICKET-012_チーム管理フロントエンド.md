## 概要

TICKET-003（チーム管理 API）に対応するフロントエンド画面を実装する。チームの作成・閲覧・編集・削除、およびメンバー招待・除外 UI を提供する。

---

## 前提条件

- TICKET-003（チーム管理 API）が完了していること
- TICKET-011（フロントエンド認証基盤）が完了していること

---

## 実装スコープ

### 画面一覧

| 画面 | パス | 対象ロール |
|---|---|---|
| チーム一覧 | `/teams` | 全員（自分の所属チームのみ / admin は全件） |
| チーム詳細・メンバー一覧 | `/teams/:id` | メンバーまたは admin |
| チーム作成 | `/teams/new` | admin / teamowner |
| チーム編集 | `/teams/:id/edit` | owner 本人または admin |

### 主要コンポーネント

- `TeamList`：チーム一覧（名前・説明・メンバー数）
- `TeamDetail`：チーム詳細＋メンバー一覧テーブル
- `TeamForm`：チーム作成・編集フォーム（名前・説明）
- `MemberInvite`：ユーザー名入力でメンバー追加
- `MemberRemoveButton`：メンバー除外ボタン（owner/admin のみ表示）
