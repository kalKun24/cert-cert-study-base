## 概要

TICKET-005（タグ管理 API）・TICKET-008（タグ検索・フィルタリング API）に対応するフロントエンドを実装する。問題一覧画面にタグフィルタ UI を追加し、タグの管理画面（admin / teamowner 向け）を提供する。

---

## 前提条件

- TICKET-005（タグ管理 API）が完了していること
- TICKET-008（タグ検索・フィルタリング API）が完了していること
- TICKET-011（フロントエンド認証基盤）が完了していること
- TICKET-009（問題管理フロントエンド）が完了していること（フィルタ UI の組み込み先）

---

## 実装スコープ

### タグフィルタ UI（問題一覧への組み込み）

- タグ一覧をチェックボックスまたはチップ形式で表示
- 複数タグを選択して AND / OR フィルタリング
- 選択状態を URL クエリパラメータで管理（ブックマーク・共有可能）

### タグ管理画面

| 画面 | パス | 対象ロール |
|---|---|---|
| タグ一覧 | `/tags` | 全員（参照のみ） |
| タグ作成・編集・削除 | `/tags` 内インライン | admin / teamowner |

---

## 受け入れ条件

- [x] 問題一覧画面でタグを選択するとリアルタイムにフィルタリングされる
- [x] 複数タグを組み合わせてフィルタリングできる
- [x] タグの選択状態が URL クエリパラメータに反映され、共有・ブックマークできる
- [x] タグ管理画面でタグの作成・編集・削除ができる（admin / teamowner のみ）
- [x] タグ削除時に確認ダイアログを表示する
