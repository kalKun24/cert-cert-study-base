## 概要

現行のサイドバー型レイアウトを廃止し、Topbar 水平ナビ + コンテンツ中央寄せの「案A・学習特化型」レイアウトへ全面刷新する。
問題文などの学習コンテンツを読みやすくするため、画面幅に応じた max-width 制限と中央寄せを全ページに適用する。

---

## 背景・目的

- 現行のサイドバーがコンテンツ表示幅を圧迫し、Markdown の問題文・解説が読みづらい
- ナビゲーション要素がコンテンツと同一視野に入ることで、学習中の集中を妨げる
- 案A（学習特化型）を採用し、Topbar に水平ナビを集約することで可読性とシンプルさを両立する

---

## 受け入れ条件

### レイアウト・ナビゲーション
- [ ] サイドバーが完全に削除されており、全ページで表示されない
- [ ] Topbar（高さ 52px）が全ページに表示され、ロゴ・水平ナビ（ホーム / 問題一覧 / タグ / ユーザー管理）・ユーザーメニューを含む
- [ ] ナビリンクのアクティブ状態が視覚的に区別できる（Teal 600 アンダーラインまたはハイライト）
- [ ] モバイル（≤768px）ではハンバーガーメニューに切り替わり、タップで開閉できる

### コンテンツ幅・中央寄せ
- [ ] 問題詳細・問題作成・問題編集ページ: コンテンツが max-width 720px で中央寄せされる
- [ ] 問題一覧・ホームダッシュボードページ: コンテンツが max-width 860px で中央寄せされる
- [ ] ユーザー管理・タグ管理などの管理テーブルページ: コンテンツが max-width 1000px で中央寄せされる
- [ ] いずれのページも左右に適切な padding（モバイル: 16px、デスクトップ: 24px 以上）が確保されている

### カラーシステム
- [ ] プライマリカラーが Teal 600（`#00897b`）に統一されている（ボタン・リンク・アクティブ状態）
- [ ] ページ背景が Grey 50（`#fafafa`）、カード背景が White（`#ffffff`）に統一されている
- [ ] 本文テキストが Grey 800（`#424242`）、ミュートテキストが Grey 600（`#757575`）に統一されている
- [ ] ボーダー半径が 6〜8px で統一されている

### アクセシビリティ
- [ ] Topbar のキーボードナビゲーション（Tab / Enter / Escape）が正常に動作する
- [ ] モバイルメニューの開閉が aria-expanded / aria-controls で適切にマークアップされている
- [ ] フォーカスインジケーターが全インタラクティブ要素で可視化されている
- [ ] テキストと背景のコントラスト比が WCAG AA 基準（4.5:1 以上）を満たす

### 既存機能の保持
- [ ] 全ページで既存の機能（問題 CRUD・タグ検索・コメント・認証）が正常に動作する
- [ ] 既存の CSS クラス名の変更によってスタイルが崩れるページがない
