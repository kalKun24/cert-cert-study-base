## 概要

問題作成画面（QuestionCreatePage）のMarkdownエディタを画面いっぱいに広げ、入力しやすくする。
メタ情報（タイトル・タグ・ステータス・ボタン）を上部スティッキーバーに集約し、エディタ領域を最大化する（案B: メタバー1行型）。

---

## 背景・目的

現状のエディタは `max-width: 720px` の中央寄せコンテナ内に高さ 300px 固定で配置されており、
Markdown の入力・プレビューに十分なスペースがない。
特に問題文・解答・解説など長文を入力するユースケースで使いにくいという課題がある。

---

## ワイヤーフレーム（案B: メタバー1行型）

```
┌─────────────────────────────────────────────────────────┐
│ META BAR (sticky)                                       │
│ [タイトル入力 (flex:1)     ] [タグ: A,C ▼] [draft▼] [保存] [取消] │
├─────────────────────────────────────────────────────────┤
│ [問題文] [解答] [解説] [議論点メモ]  ← タブ（全幅）    │
├─────────────────────────────────────────────────────────┤
│ MDEditor toolbar                                        │
├──────────────────────────────┬──────────────────────────┤
│                              │                          │
│   テキスト入力エリア          │   Markdown プレビュー   │
│                              │                          │
│   ↕ calc(100vh - メタバー)   │   ↕ 同左               │
│                              │                          │
└──────────────────────────────┴──────────────────────────┘
```

タグドロップダウン展開時:
```
│ META BAR                                                │
│ [タイトル...]  [タグ: A,C ▼]  [draft▼]  [保存] [取消] │
│               ┌────────────────┐                       │
│               │ ☑ タグA        │                       │
│               │ ☐ タグB        │                       │
│               │ ☑ タグC        │                       │
│               └────────────────┘                       │
```

## 受け入れ条件

- [ ] メタバー（タイトル・タグ・ステータス・保存/キャンセルボタン）が画面上部に sticky 固定されている
- [ ] タイトル入力欄がメタバー内に横幅 flex:1 で配置されている
- [ ] タグ選択がドロップダウン形式で複数選択できる（選択中タグ名をカンマ区切りでボタンに表示）
- [ ] ステータス選択がメタバー内に配置されている
- [ ] MDEditor が `calc(100vh - メタバー高さ)` 相当で縦方向に画面いっぱいに伸張している
- [ ] MDEditor が全幅（`max-width` 制限なし）で横方向に広がっている
- [ ] タブ（問題文 / 解答 / 解説 / 議論点メモ）の切り替えが引き続き機能する
- [ ] モバイル（768px 以下）ではメタバーが2行に折り返し、エディタが縦1列（プレビューなし）になる
- [ ] バリデーション（タイトル必須・問題文必須）が引き続き機能する
