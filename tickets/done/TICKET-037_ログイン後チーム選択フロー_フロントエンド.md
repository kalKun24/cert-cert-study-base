## 概要

ログイン後、ユーザーのチーム所属状況に応じた画面分岐を実装する。
また、Topbar にアクティブチームラベルとチーム切り替えプルダウンを追加する。

---

## 背景・目的

チーム制に移行したことで、ログイン直後に「どのチームで作業するか」を選ぶフローが必要になった。

---

## 受け入れ条件

- [x] ログイン後、チーム所属状況に応じて以下の3パターンに分岐する
  - **どのチームにも招待されていない**: 「どのチームにも招待されていません」画面を表示
  - **招待のみ・所属チームなし**: 招待チームリストを表示し、参加/拒否を選択できる
  - **所属チームあり**: 最後にアクセスしたチーム（またはデフォルトチーム）のホームへ遷移
- [x] Topbar に「現在のチーム名」ラベルを表示する
- [x] Topbar にチーム切り替えプルダウンを追加し、所属全チームを一覧できる
- [x] チームオーナーには Topbar に「チームを作成」メニューが表示される
- [x] チームから脱退できる UI をチーム設定画面またはプロフィール画面に追加する

---

## サブチケット（コミット単位）

- [x] `feat(context): TeamContext を作成し現在のチーム状態をグローバル管理`
- [x] `feat(page): ログイン後チーム分岐ルーティングを実装`
- [x] `feat(page): 招待なし画面・招待リスト画面を作成`
- [x] `feat(component): Topbar にアクティブチームラベル + 切り替えプルダウンを追加`
- [x] `feat(component): チームオーナー向け「チームを作成」メニューを追加`
- [x] `feat(page): チーム脱退 UI を実装`
