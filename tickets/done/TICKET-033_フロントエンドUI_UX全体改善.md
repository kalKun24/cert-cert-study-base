## 概要

現状のフロントエンドUIにはデザイントークンの不整合・未定義CSSクラスによるレイアウト崩れ・MDEditorとカラーパレットの乖離など複数の問題がある。これらを修正しつつ、解答折りたたみ・ダッシュボード化・アクセシビリティ改善を含む全体的なUI/UX向上を行う。

---

## 背景・目的

- UX Architect・UI Designer による現状調査で、バグレベルの問題（未定義CSSクラス・WCAG未達カラートークン）と高インパクトな改善点が特定された
- 学習フロー（問題を解いてから解答を確認する）に適したUI構造になっていない
- デザイントークンの不整合により一貫したビジュアルが実現できていない

---

## 受け入れ条件

- [ ] デザイントークンの不整合が解消され、コントラスト比がWCAG AA（4.5:1）を満たす
- [ ] 問題作成・編集フォームのレイアウトが正しく表示される
- [ ] MDEditorがダークティールパレットと視覚的に一致する
- [ ] 解答・解説セクションが折りたたみ式になり、デフォルトで閉じている
- [ ] ホーム画面がダッシュボードとして機能する（統計カード・最近の問題）
- [ ] サイドバーにアイコンとグルーピングラベルが追加されている
- [ ] TagChipの選択状態がアクセシブルなスタイルになっている
- [ ] スケルトンローディングが主要画面に追加されている
- [ ] モバイルでテーブルがカードリスト表示になる
- [ ] キーボードナビゲーションが全コンポーネントで機能する

---
